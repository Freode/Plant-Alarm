## 요약
- 알람을 여러개 설정해도 쉽게 일어나지 못하는 사람들을 위해
- 알람을 종료할려면, 지뢰찾기와 같은 미션을 해결할 수 있도록 구현했습니다.
- 미션 성공 여부에 따라 식물을 키울 수 있도록 성장 기능도 추가했습니다.
